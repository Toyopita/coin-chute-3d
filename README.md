# コイン計算機用シュート

## 📐 設計仕様

### 概要
コイン計算機の排出口に設置し、コインを袋に誘導するシュートです。
2分割構造で、Bambu Lab A1（造形サイズ: 256×256×256mm）で印刷可能です。

### サイズ
- **上部受け口**: 240mm × 315mm
- **下部出口**: 直径100mm
- **各パーツの高さ**: 60mm
- **壁厚**: 2mm
- **全体の高さ**: 120mm（2パーツ組み立て後）
- **傾斜角度**: 約40度（コインがスムーズに滑り落ちる設計）

### 構造
1. **上部パーツ** (`coin_chute_upper.stl`)
   - 上部: 長方形（240×315mm）- コイン計算機の排出口に適合
   - 下部: 楕円形（180×220mm）- 下部パーツとの接続部
   - 4箇所のスナップフィット突起付き

2. **下部パーツ** (`coin_chute_lower.stl`)
   - 上部: 楕円形（180×220mm）- 上部パーツとの接続部
   - 下部: 円形（直径100mm）- 袋の口に適合
   - 4箇所のスナップフィット穴付き

## 🖨️ プリント設定（Bambu Lab A1推奨）

### 基本設定
- **材料**: PLA または PETG
- **ノズル温度**:
  - PLA: 200-220°C
  - PETG: 230-250°C
- **ベッド温度**:
  - PLA: 60°C
  - PETG: 80°C
- **レイヤー高さ**: 0.2mm
- **壁厚**: 3-4層（2.4-3.2mm）
- **充填率**: 15-20%
- **充填パターン**: Grid または Gyroid
- **サポート**: 不要（設計上サポートなしで印刷可能）
- **接着**: ブリム推奨（特に上部パーツ）

### プリント方向
- **上部パーツ**: 楕円形の面を下にして配置
- **下部パーツ**: 円形の面を下にして配置

### プリント時間（概算）
- **上部パーツ**: 約4-5時間
- **下部パーツ**: 約3-4時間
- **合計**: 約7-9時間

## 🔧 組み立て方法

1. **上部パーツと下部パーツの接続**
   - 上部パーツの下部（楕円形）と下部パーツの上部（楕円形）を合わせる
   - 4箇所のスナップフィット突起を穴に差し込む
   - 「カチッ」という音がするまで押し込む

2. **接着（オプション）**
   - より強固にしたい場合は、接合部に接着剤を塗布
   - 推奨: プラスチック用瞬間接着剤 または 2液性エポキシ

## 📦 使用方法

1. コイン計算機の排出口に上部パーツの長方形開口部を設置
2. 下部の円形出口（直径100mm）に袋の口を当てる
3. 袋を手で持って受ける
4. コインがシュートを滑り落ちて袋に入る

## 💡 Tips

- **滑りやすさ**: PLAは表面が滑らかなので、コインがスムーズに滑り落ちます
- **耐久性**: 頻繁に使う場合はPETGの方が耐久性が高いです
- **カスタマイズ**: `generate_stl.py`のパラメータを変更すれば、サイズを調整できます
  - `TOP_WIDTH`, `TOP_DEPTH`: 上部サイズ
  - `BOTTOM_DIAMETER`: 下部出口サイズ
  - `HEIGHT_PER_PART`: 各パーツの高さ

## 🔄 再生成方法

サイズを変更したい場合:

```bash
cd ~/3d_models/coin_chute
# パラメータを編集
nano generate_stl.py

# STL再生成
python3 generate_stl.py
```
